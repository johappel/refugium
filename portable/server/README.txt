Optionales Server-Binary

Bevorzugte Variante:
- miniserve.exe (Windows x64)

Der Start-Launcher sucht zuerst nach:
- portable\server\miniserve.exe
- danach nach portable\server\caddy.exe
- sonst faellt er auf serve-refugium.ps1 zurueck

Automatischer Download fuer Entwickler:
- scripts\download-miniserve.ps1
- oder npm run download:miniserve
